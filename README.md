## 🔍 Problem Overview

### 1️⃣ Post Engagement Analyzer
Calculates total likes and determines whether a post is Viral or Normal.

### 2️⃣ Medicine Stock Alert
Checks stock levels and generates alert if stock is low.

### 3️⃣ Rainfall Adequacy Checker
Computes average rainfall and evaluates adequacy for agriculture planning.

### 4️⃣ Duplicate Account Detection
Uses set logic to detect duplicate usernames.

### 5️⃣ Appointment Eligibility Checker
Checks if patient age qualifies for appointment booking.

### 6️⃣ Premium Crop Price Filter
Filters crop prices above ₹2000.

### 7️⃣ Application Health Checker
Classifies system health based on error count.

### 8️⃣ Daily Transaction Limit Checker
Validates transaction against ₹50,000 daily limit.

### 9️⃣ Attendance Eligibility System
Calculates attendance percentage and determines exam eligibility.

### 🔟 Smart Electricity Bill Analyzer
Calculates electricity bill using slab rates and classifies usage.
